# Migration Guide

This folder mirrors the OpenCode-facing distribution surface for AI Dev Kit.

Use it when:
- packaging the repo for OpenCode
- checking how prompts and agent metadata are organized
- keeping the OpenCode surface aligned with the main repo

Main entrypoints:
- `opencode.json`
- `index.ts`
- `plugins/`
- `tools/`
- `instructions/`
- `prompts/`
