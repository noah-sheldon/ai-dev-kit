# OpenCode Plugin — AI Dev Kit

Production-ready AI dev workflow scaffold as an OpenCode plugin.

## Structure

```
.opencode/plugins/
├── package.json          # npm package metadata
├── tsconfig.json         # TypeScript configuration
├── src/
│   └── index.ts          # Plugin entry point (tools, events, config)
└── dist/                 # Compiled output (generated by npm run build)
```

## Local Development

```bash
cd .opencode/plugins
npm install
npm run build
```

## Usage

The parent `.opencode/opencode.json` already references this plugin via:

```json
{
  "plugin": ["./plugins"]
}
```

## Published

To publish as an npm package:

```bash
npm publish --access public
```

Users then install via `npm install opencode-ai-dev-kit` and add to their `opencode.json`:

```json
{
  "plugin": ["opencode-ai-dev-kit@1.0.0"]
}
```
