# AI Dev Kit — Gemini Extension

Production-ready AI dev workflow scaffold with 50+ skills, 19 agents, hooks, rules, validation, and MCP configs.

## What's available

- **50+ skills** — Python, TypeScript, ML, infra, security, testing, deployment, and more.
- **19 agents** — Planner, architect, code reviewer, security reviewer, TDD guide, and others.
- **Hooks & rules** — Automated quality gates, git workflows, and style enforcement.
- **MCP configs** — Pre-configured tool integrations.

## Usage

Use the skills and agents defined in `./skills/` and `./agents/` to delegate specialized work. Follow the core workflow principles:

1. Plan complex work first.
2. Write tests before implementation.
3. Review code immediately after changes.
4. Keep docs, manifests, and templates in sync.

See `AGENTS.md` in the repo for complete agent descriptions, workflow guidelines, and project structure.
