# AI Dev Kit — Gemini Extension

Production-ready AI dev workflow scaffold with 59 skills, 33 agents, hooks, and rules.

## What's available

- **59 skills** — Python, TypeScript, ML, infra, security, testing, deployment, data pipelines, API design, and more.
- **33 agents** — Planner, architect, code reviewer, security reviewer, TDD guide, ML engineer, data engineer, and 11 debt analyzers.
- **41 commands** — Slash-command shims for build, workflow, git, hooks, context, learning, and diagnostics.
- **Hooks & rules** — Automated quality gates, git workflows, and style enforcement (Python, TypeScript, web).

## Core Principles

1. **Agent-first** — delegate domain work to the right specialist.
2. **Test-driven** — write tests before implementation when behavior changes.
3. **Security-first** — validate inputs, avoid unsafe defaults, never hardcode secrets.
4. **Plan-before-execute** — break complex work into phases with the planner.

## Usage

Use the skills and agents defined in `./skills/` and `./agents/` to delegate specialized work.
